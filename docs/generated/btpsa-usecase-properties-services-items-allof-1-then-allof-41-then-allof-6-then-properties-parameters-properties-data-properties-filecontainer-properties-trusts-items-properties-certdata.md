## certData Type

`string`

## certData Constraints

**pattern**: the string must match the following regular expression:&#x20;

```regexp
^-----BEGIN CERTIFICATE-----\s[A-Za-z0-9+/\s]+={0,2}\s-----END CERTIFICATE-----$
```
